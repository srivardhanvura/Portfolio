Shipping Application

An Android application to ship goods over the country.

Live Link

Project Overview

Developed a courier shipping application using Java and XML. Implemented Firebase for robust authentication and user profile data storage, including past bookings and courier images. Embedded Google Maps for tracking the locations of senders and receivers.

The users of the application can send couriers from any part of the country to any other location within India. The users can select the sender and destination locations via placing a pin on the map or by typing the location. Then they will select the specifications of the parcel and the application will quote a price for the courier.
