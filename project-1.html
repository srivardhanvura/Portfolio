CourseWizard

A comprehensive course enrollment and management web application

Live Link

Project Overview

Coursewizard is a fully functional course management portal that can be used by students, teachers of any University. The website was developed using Java, Spring MVC, PostgreSQL, and Thymeleaf. Engineered RESTful APIs to accommodate diverse use cases and instituted role-based access control, enhancing system versatility for both students and educators.

The website has a special role- ADMIN which can be used to enroll new students and instructors, add a new course and delete any users in the system. The users with role- INSTRUCTOR can edit their courses attributes like changing the class room, location, strength, or make the course inactive. The users with role- STUDENT can enroll or drop any courses in their departments. They can rate a course and look at the reviews of the courses given by students who have taken it previously. All the users can update their profile.
